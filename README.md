## Program Details
The MotorPH Payroll System is a Java-based payroll application that processes employee attendance and salary records.

The program includes a login system that provides different access levels for payroll staff and employees.

The program reads employee information and attendance data from CSV files. It calculates the total hours worked by employees and determines their payroll for two cutoff periods: 1–15 and 16–end of the month.

The system automatically computes the following government deductions based on monthly gross salary:
- SSS
- PhilHealth
- Pag-IBIG
- Withholding Tax

The government deductions are calculated based on the total gross salary per month and it's only deducted on the 2nd Gross Salary.

There are two system roles:

Payroll Staff
- Process payroll for a single employee
- Process payroll for all employees

Employee
- View basic employee information

The program uses Java file reading and time libraries to compute working hours and generate payroll reports.
